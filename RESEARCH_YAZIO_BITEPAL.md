### 🧮 Cálculo de Calorías (Fórmula Mifflin-St Jeor)

```
TMB (Mujeres) = 10 × peso(kg) + 6.25 × altura(cm) - 5 × edad - 161
TMB (Hombres) = 10 × peso(kg) + 6.25 × altura(cm) - 5 × edad + 5

TDEE = TMB × factor_actividad
  - Sedentario: 1.2
  - Ligero: 1.375
  - Moderado: 1.55
  - Activo: 1.725
  - Muy activo: 1.9

Calorías objetivo = TDEE - déficit (500-1000 para pérdida)
```

### 3. CÁLCULO DE CALORÍAS (Implementación)

```javascript
// Cálculo TMB (Mifflin-St Jeor)
function calculateTMB(gender, weight, height, age) {
  if (gender === 'male') {
    return 10 * weight + 6.25 * height - 5 * age + 5;
  } else {
    return 10 * weight + 6.25 * height - 5 * age - 161;
  }
}

// Factor de actividad
const ACTIVITY_FACTORS = {
  sedentary: 1.2,
  light: 1.375,
  moderate: 1.55,
  active: 1.725,
  very_active: 1.9
};

// Déficit calórico por velocidad
const DEFICIT_BY_SPEED = {
  slow: 250,      // 0.25 kg/semana
  moderate: 500,  // 0.5 kg/semana
  fast: 1000      // 1 kg/semana
};

// Cálculo final
function calculateDailyCalories(user) {
  const tmb = calculateTMB(user.gender, user.weight, user.height, user.age);
  const tdee = tmb * ACTIVITY_FACTORS[user.activityLevel];
  const deficit = DEFICIT_BY_SPEED[user.speed];
  
  return {
    maintenance: Math.round(tdee),
    target: Math.round(tdee - deficit),
    protein: Math.round(user.weight * 2), // 2g por kg
    carbs: Math.round((tdee - deficit) * 0.4 / 4), // 40% en gramos
    fat: Math.round((tdee - deficit) * 0.3 / 9)    // 30% en gramos
  };
}
```
