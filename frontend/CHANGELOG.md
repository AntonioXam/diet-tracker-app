# Changelog

Todas las modificaciones notables a este proyecto serán documentadas en este archivo.

## [2.0.0] - 2026-03-01

### ✨ Nueva Funcionalidad
- **Modo oscuro completo** con toggle persistente en localStorage
- **Diseño responsive** optimizado para móvil, tablet y escritorio
- **Tarjetas de recetas mejoradas** con imágenes de la API
- **Dashboard visual** con gráficos y estadísticas
- **Lista de compras** agrupada por supermercado
- **Sistema de notificaciones** con toasts animados

### 🎨 Mejoras de Diseño
- **Rediseño completo** con gradientes y efectos glassmorphism
- **Tipografía Inter** mejorada para mejor legibilidad
- **Variables CSS** para colores y transiciones
- **Micro-interacciones** en botones y tarjetas
- **Estados de carga** animados
- **Placeholders** con colores por tipo de comida

### 📱 Experiencia de Usuario
- **Navegación intuitiva** por días de la semana
- **Progreso semanal** visual con barras de progreso
- **Distribución de calorías** por comidas
- **Feedback visual** en todas las acciones
- **Accesibilidad mejorada** con ARIA labels
- **Navegación por teclado** optimizada

### 🔧 Refactorización Técnica
- **Separación de archivos**: HTML, CSS y JavaScript independientes
- **Código modular** con funciones organizadas
- **Variables CSS** para temas claro/oscuro
- **Animaciones CSS** en lugar de JavaScript
- **Optimización de performance**
- **Mejor manejo de errores**

### 🐛 Correcciones de Bugs
- **Imágenes de recetas** ahora se muestran correctamente
- **Cálculo de calorías** actualizado con la API
- **Responsive design** en todos los componentes
- **Modales** con mejor comportamiento
- **Formularios** con validación mejorada

### 📦 Dependencias Actualizadas
- **Tailwind CSS** configurado para modo oscuro
- **Chart.js** para gráficos interactivos
- **Google Fonts** (Inter) para tipografía
- **Animaciones CSS** personalizadas

## [1.0.0] - 2026-02-28

- Lista de compras simple
